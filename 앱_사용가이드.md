# CareCheck - 안드로이드 사례관리 앱 사용 가이드

선생님의 요구사항을 반영하여 제작된 **'안드로이드 맞춤형 사례관리 앱'**의 사용 설명서입니다.

## 📱 앱 실행 방법 (미리보기)

현재 폴더에 있는 `index.html` 파일을 마우스 오른쪽 버튼으로 클릭한 뒤 **[연결 프로그램] > [Chrome] 또는 [Microsoft Edge]**를 선택하여 실행하세요.
- 실제 안드로이드 폰에서 보는 것과 같은 세련된 디자인을 확인하실 수 있습니다.

## ✨ 앱 주요 기능

1.  **명단 확인**: 전체 대상자 명단과 검색 기능을 제공합니다.
2.  **주간 체크 (가장 중요!)**: 리스트 우측의 **체크 아이콘**을 누르면 이번 주 상담 완료 여부가 즉시 업데이트됩니다. (로컬 데이터에 저장됩니다.)
3.  **메모 작성**: 리스트의 **문서 아이콘**을 누르면 해당 대상자에 대한 상담 메모를 적고 저장할 수 있습니다.

## 🔗 구글 시트 실제 데이터와 연동하기

현재 앱은 테스트용 가짜 데이터로 작동하고 있습니다. 실제 선생님의 구글 시트와 연결하려면 아래 과정을 딱 한 번만 수행하시면 됩니다.

### [1단계] 브릿지 코드 복사
아래 코드를 전체 복사하세요.

```javascript
/**
 * CareCheck Google Sheets Bridge
 */
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var headers = data[0];
  var rows = data.slice(1);
  var result = rows.map(function(row, index) {
    var obj = { rowId: index + 2 };
    headers.forEach(function(header, i) { obj[header] = row[i]; });
    return obj;
  });
  return ContentService.createTextOutput(JSON.stringify(result)).setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  var params = JSON.parse(e.postData.contents);
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
  var colIndex = headers.indexOf(params.columnName) + 1;
  if (colIndex > 0) {
    sheet.getRange(params.rowId, colIndex).setValue(params.value);
    return ContentService.createTextOutput("Success").setMimeType(ContentService.MimeType.TEXT);
  }
  return ContentService.createTextOutput("Column not found").setMimeType(ContentService.MimeType.TEXT);
}
```

### [2단계] 구글 시트에 붙여넣기
1.  관리하시려는 **구글 시트**를 엽니다.
2.  상단 메뉴에서 **[확장 프로그램] > [Apps Script]**를 클릭합니다.
3.  기존에 있는 코드를 모두 지우고, 위에서 복사한 코드를 붙여넣습니다.
4.  왼쪽 상단 프로젝트 이름을 "CareCheck_Bridge" 등으로 저장합니다.

### [3단계] 웹 앱으로 배포
1.  오른쪽 상단 주황색 **[배포] > [새 배포]**를 누릅니다.
2.  유형 선택(톱니바퀴)에서 **[웹 앱]**을 선택합니다.
3.  설정:
    - 설명: CareCheck 연동
    - 다음 사용자 권한으로 실행: **나 (내 이메일 주소)**
    - 액세스 권한이 있는 사용자: **모든 사용자 (Anyone)**
4.  [배포] 버튼을 누르고 권한 승인을 완료합니다.
5.  생성된 **웹 앱 URL**을 복사해서 저에게 알려주시면 앱에 최종 적용해 드립니다!
